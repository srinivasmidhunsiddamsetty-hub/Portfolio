# Srinivas Midhun — Portfolio

Personal portfolio website. Hand-coded in HTML, CSS, and vanilla JS — no framework, no build step.

## Structure

- `index.html` — Homepage
- `about.html` — About Me
- `paypal.html`, `smartcharge.html`, `campus-copilot.html`, `designiq.html` — Case studies
- `images/` — All photography, mockups, diagrams, and videos
- `SrinivasMidhun_Resume.pdf` — Resume

## Stack

- HTML5, CSS3 (custom properties, grid, flex, `clamp()`)
- Vanilla JavaScript (theme toggle, scroll-spy, video progress)
- DM Sans via Google Fonts
- No build pipeline. Open `index.html` to develop locally.

## Deploy

Hosted on Vercel, connected to this repo. Every push to `main` auto-deploys.
